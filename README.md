# MicroProxy

Легковесный кэширующий HTTP-прокси с графическим интерфейсом для ускорения интернета на старом железе.

## Возможности

- Кэширует HTTP-запросы на диск
- Многопоточная обработка клиентов
- Минимальное потребление ресурсов (~5 МБ RAM)
- Ускоряет повторные запросы к одним и тем же сайтам в 10-100 раз
- Простой графический интерфейс с одной кнопкой

## Сборка

### Требования
- Go 1.21+
- Linux: `sudo apt install gcc libgl1-mesa-dev libx11-dev libxcursor-dev libxrandr-dev libxi-dev libxinerama-dev libxext-dev libxxf86vm-dev`

### Компиляция
```bash
go build -o microproxy
```

## Запуск

```bash
./microproxy
```

Откроется окно с кнопкой "Включить прокси". Нажми её — прокси запустится на порту 8080.

## Использование

Настрой браузер на использование прокси:
- HTTP прокси: `127.0.0.1`
- Порт: `8080`

Или через командную строку:
```bash
curl -x http://127.0.0.1:8080 http://example.com/
```

## Производительность

- Первый запрос: идёт в интернет
- Повторные запросы: мгновенно из кэша
- Потребление памяти: <10 МБ
- Поддерживает параллельные соединения
